# Byte Parts and Its Terminology

## 1. What is a Byte?
- A **byte** is a unit of digital information used in computers.
- 1 byte = **8 bits**.
- Each bit is the smallest unit of data (0 or 1).

---

## 2. Parts of a Byte
1. **Bit (b)**  
   - Smallest unit of data in a computer.  
   - Can be 0 or 1.  

2. **Nibble**  
   - Half a byte = 4 bits.  
   - 1 byte = 2 nibbles.  
   - Often used in hexadecimal representation.  

3. **Byte (B)**  
   - 8 bits = 1 byte  
   - Can store values 0–255 (unsigned).  

4. **Word**  
   - A group of bytes (depends on CPU architecture).  
   - Example: 16-bit CPU → 1 word = 2 bytes  
             32-bit CPU → 1 word = 4 bytes  

---

## 3. Common Terminology
| Term   | Size       | Description |
|--------|------------|-------------|
| Bit (b)| 1 bit      | Smallest data unit (0 or 1) |
| Nibble | 4 bits     | Half a byte, used in Hex |
| Byte (B)| 8 bits    | Stores 1 character or 0–255 value |
| Word   | 2–4 bytes | Depends on CPU architecture |
| Kilobyte (KB)| 1024 bytes | Memory or storage unit |
| Megabyte (MB)| 1024 KB | Memory or storage unit |
| Gigabyte (GB)| 1024 MB | Memory or storage unit |
| Terabyte (TB)| 1024 GB | Memory or storage unit |

---

## 4. Summary Notes
- 1 byte = 8 bits = 2 Hex digits = 1 character (ASCII).  
- Nibbles are useful for **Hexadecimal conversions**.  
- Words depend on **CPU architecture** (16-bit, 32-bit, 64-bit).  
- Understanding byte terminology is crucial for **memory management, programming, and cybersecurity**.
